# Aero Tech
## The Problem

Many weather apps show current conditions, but do not help users plan for short unpredictable windows of good weather during a busy day. Show hour-by-hour conditions and highlight ideal time slots for outdoor tasks like walking the dog, running, or commuting without getting caught in rain or poor visibility.

## Summary

An hour-by-hour weather calendar that shows ideal conditions and time slots for outdoor tasks.

# My Implementation
## API Usage
Using the API, OpenWeatherMap + an API key
